Batch Script ডাউনলোড করুন

নিচে দেওয়া বাটনে ক্লিক করে ব্যাচ ফাইলটি ডাউনলোড করুন যা আপনার উইন্ডোজ সিস্টেম অ্যাকটিভেট করতে সাহায্য করবে।

Batch Script ডাউনলোড করুন

Instructions:

English: To activate Windows, click the "Download Batch Script" button. Once downloaded, run the file to activate your Windows system.

বাংলা: উইন্ডোজ অ্যাকটিভেট করতে, "Download Batch Script" বাটনে ক্লিক করুন। ফাইলটি ডাউনলোড হলে, তা রান করে আপনার উইন্ডোজ অ্যাকটিভেট করুন।

যদি ব্যাচ স্ক্রিপ্ট কাজ না করে, তবে আপনি CMD ব্যবহার করে উইন্ডোজ ম্যানুয়ালি অ্যাকটিভেট করতে পারেন:

নীচে উল্লেখিত পদক্ষেপগুলো অনুসরণ করুন:

1. Windows key প্রেস করুন এবং "cmd" টাইপ করুন।
2. Command Prompt এ রাইট ক্লিক করুন এবং Run as administrator নির্বাচন করুন।
3. Command Prompt-এ নিচের কমান্ডগুলো টাইপ করুন এবং Enter প্রেস করুন:


   slmgr /skms kms.lotro.cc
   slmgr /ato




4. আপনার উইন্ডোজ এখন অ্যাকটিভেট হয়ে যাবে INSHA ALLAH !
